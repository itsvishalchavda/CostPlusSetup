How to Install "CostPlus (Product Costing Estimator v1.1.1)"

1. Install CostPlusSetup.msi
2. After Install, you will have shortcut icon
on desktop double click it and run. if there is 
missing DLL Error than we need to install 
following additional runtime packages.
3. First Install vcredist_x86_100.exe (Visual C++ 2010)
4. Than Install vcredist_x86_140.exe (Visual C++ 2015)
5. Now run the softwares from desktop icon.
6. During printing reports, if u get any error than
install ReportViewer2005.exe (You may need to install .Net 3.5)
7. Additionally install .Net 4.5 (or 4) and .Net 3.5 SP1 if needed.
8. you can download the .Net from following official links.
.Net 3.5 -> https://www.microsoft.com/en-in/download/details.aspx?id=25150
.Net 4.0 -> https://www.microsoft.com/en-in/download/details.aspx?id=17718
